# MCD EcoRide

Ce document contient le Modèle Conceptuel de Données (MCD) du projet EcoRide.

Le MCD permet de représenter :
- les utilisateurs ;
- les trajets ;
- les réservations ;
- les véhicules ;
- les avis ;
- les préférences utilisateurs.

Il permet également d’anticiper la future mise en place d’une base de données relationnelle pour le projet.

Le schéma complet du MCD est présent dans le dossier de conception du projet.
